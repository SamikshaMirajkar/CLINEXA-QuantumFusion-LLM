# CLINEXA / QuantumFusion-LLM

CLINEXA is a multimodal AI-powered clinical intelligence system for analyzing Chest X-rays, ECG signals, and patient symptoms to generate disease predictions, confidence scores, severity levels, heatmaps, and AI-generated clinical reports.

## Features
- Patient registration with live photo capture
- Chest X-ray upload
- ECG upload
- Symptom analysis
- Disease prediction
- Confidence and severity scoring
- Heatmap visualization
- Voice-assisted report summary
- PDF clinical report generation

## Tech Stack
- Frontend: React, Framer Motion
- Backend: FastAPI, Python
- Models: ResNet18, 1D CNN, Text Analysis, Quantum-inspired Fusion
- Libraries: PyTorch, NumPy, Pillow, jsPDF
